This paper presents capabilities for modifying parameters in real-time within Mesa-based agent-based model simulations. Mesa is an agent-based modeling framework written in Python, and our work enhances the interactive exploration of complex system dynamics by enabling researchers to adjust model parameters during simulation runs.
